# 🏎️ F1 Constructor Performance Dashboard | Power BI

An interactive Power BI dashboard analyzing Formula 1 constructor performance from 2022 to 2024. Track championship standings, win rates, reliability metrics, and race-day correlations in a single, dynamic view.

---

## 📊 Key Metrics at a Glance

| Metric | Value |
|--------|--------|
| **Constructor Wins** | 1,128 |
| **Constructor Reliability** | 99.91% |
| **Championship Titles** | 3 |

---

## 📈 Dashboard Visuals

The Power BI report includes the following interactive visuals:

| Visual | Description |
|--------|-------------|
| **Final Constructor Championship Position by Year** | Year-over-year championship rank tracking per constructor |
| **Constructor Win Rate (%) by Name** | Win percentage breakdown by team |
| **Current Season Reliability vs Target** | Actual reliability performance against season targets |
| **Championship Status** | Real-time title contender and clinch status |
| **Total Points Scored vs Grid Start Position** | Correlation analysis between qualifying position and race points |

---

## 🛠️ Built With

| Tool | Purpose |
|------|---------|
| **Power BI Desktop** | Data modeling, DAX calculations, and interactive visualization |
| **Power Query** | Data extraction and transformation |
| **DAX** | Custom measures for win rates, reliability scoring, and year-over-year comparisons |

---

## 🚀 Purpose

This project provides a clean, visual summary of F1 constructor statistics for analysts, fans, and team strategists. It's designed to support quick insights and performance comparisons across multiple seasons.

---

## 📌 Key Takeaways

- **Reliability is elite** – A 99.91% reliability rate reflects near-perfect engineering.
- **Win volume matters** – Over 1,100 constructor wins captured.
- **Championship pedigree** – Three titles highlight sustained excellence.
